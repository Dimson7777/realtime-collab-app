# Syncly — Real-Time Collaborative Workspace

> A production-style real-time collaboration platform with shared workspaces, live presence, and instant synchronization.

---

## 🚀 Overview

Syncly is a modern SaaS-inspired web application designed to simulate real-world collaborative tools.  
It enables multiple users to interact in real-time through shared rooms, live cursors, and synchronized state.

Built with a strong focus on performance, scalability, and clean architecture.

---

## ✨ Features

- Real-time multi-user collaboration  
- Live presence tracking (online users)  
- Live cursors with smooth movement  
- Built-in room-based chat  
- Collaborative board (sticky notes)  
- Auto-save & state persistence  
- Basic version history  
- Authentication (login/signup)  
- Clean session handling (no data leaks)  
- Fully responsive UI

## 🔐 Authentication & Data Isolation

- Each user sees only their own data  
- State resets on logout  
- Fresh data loaded on login  
- No shared state between users
