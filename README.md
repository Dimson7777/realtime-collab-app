# Syncly — Real-Time Collaborative Workspace

A modern SaaS-style web application that enables teams to collaborate in real-time through shared rooms, live chat, and interactive boards.

Built with a focus on performance, clean UX, and production-like behavior, Syncly simulates how real collaborative tools (like Notion or Slack) work under the hood.

---

## ✨ Core Features

* ⚡ Real-time collaboration across multiple users
* 👥 Live presence (see who is online)
* 🖱️ Live cursors with smooth motion & labels
* 💬 Built-in chat system inside rooms
* 🧠 Sticky notes / collaborative board
* 💾 Auto-save functionality
* 🕘 Version history (basic)
* 🔐 Authentication (login/signup)
* 🧹 Clean user sessions (no data leaks between users)
* 📱 Fully responsive UI

## 🔐 Authentication & Data Isolation

* Each user sees only their own data
* Full state reset on logout
* Fresh data loaded on every login
* No shared state between users
